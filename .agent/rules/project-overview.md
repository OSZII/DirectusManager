# Project Overview

Directus Manager is a desktop application built with **Electron** and **Vue 3** designed to manage multiple Directus instances.

## Primary Goals
- Simplify the process of syncing configurations between instances (Pull/Push).
- Manage Directus configurations via Git version control.
- Provide a centralized GUI for instance configuration.
